Comic-Web

Comic-web is a markup language for describing comics, as well as a visualizer library for this format.

Layers

Every word that is capitalized and/or followed by a new line signals a layer in the panel. 
MAIN STREET

The visualizer searches an image that goes by the name of the layer in the designited folder, only the file names should be lowercase and use dashes instead of spaces as separators, (so 'MAIN STREET', becomes 'main-street.svg'.
